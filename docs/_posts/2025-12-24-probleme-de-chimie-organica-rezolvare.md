**ALCANI**

### 1. **Care este formula moleculară a unui alcan care are masa molară M = 44?**

Rezolvare: <span class="math inline">\(C_{n}H_{2n + 2}\)</span> este formula generală a unui alcan și are <span class="math inline">\(M = 12n + 2n + 2 = 14n + 2\)</span>

Atunci: <span class="math inline">\(14n + 2 = 44 \rightarrow 14n = 42 \rightarrow n = 3\)</span>

<div class="math display">\[\rightarrow C_{3}H_{8}\ este\ formula\ moleculară\ a\ alcanului,\ propan\]</div>

### 2. **Care este formula moleculară a alcanului dacă prin arderea a 88 g alcan se formează 264 g bioxid de carbon?**

Rezolvare: <span class="math inline">\(C_{n}H_{2n + 2} + {\frac{3n + 1}{2}O}_{2} \rightarrow n{CO}_{2} + (n + 1)H_{2}O\)</span>

Atunci: la (14n+2) g alcan...................44n g bioxid de carbon  
88 g alcan.............................264 g bioxid de carbon  
264<span class="math inline">\(\bullet\)</span>(14n+2)=44<span class="math inline">\(n \bullet\)</span> 88<span class="math inline">\(\rightarrow\)</span> 84n+12= 88n<span class="math inline">\(\rightarrow\)</span>12 = 4n<span class="math inline">\(\rightarrow\)</span>n =3,
deci <span class="math inline">\(C_{3}H_{8}\ este\ formula\ moleculară\ a\ alcanului,\ propan\)</span>

### 3. **Care este formula moleculară a unui alcan dacă prin dehidrogenarea a 90 g alcan se formează 67,2 litri hidrogen, măsurat în condiții normale?**

Rezolvare: <span class="math inline">\(C_{n}H_{2n + 2} \rightarrow C_{n}H_{2n} + nH_{2}\)</span>

<span class="math inline">\((14n + 2)g\ \ alcan\ldots\ldots\ldots\ldots\ldots\ldots\ldots.22,4 \bullet n\ \ L\ \ hidrogen\)</span>

<div class="math display">\[90\ g\ \ alcan\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots.67,2\ L\ \ hidrogen\]</div>

<div class="math display">\[\rightarrow 14n + 2 = 30 \rightarrow n = 2 \rightarrow C_{2}H_{6\ }este\ formula\ moleculară\ a\ alcanului,\ etan\]</div>

### 4. **Cât negru de fum se obține prin oxidarea a 112 L metan, măsurat în condiții normale?**

Rezolvare: <span class="math inline">\({CH}_{4} + O_{2} \rightarrow C + 2H_{2}O\)</span>

<div class="math display">\[22,4\ L\ \ {CH}_{4}\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots 12\ g\ C\]</div>

<div class="math display">\[112\ L\ \ {CH}_{4}\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots x\ g\ C\  \rightarrow x = 60\ g\ \ C\]</div>

### 5. **Care este volumul gazului de sinteză(c.n), obținut prin procedeul arderii incomplete a 160 g metan?**

<div class="math display">\[160g\ \ \ {CH}_{4}\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots x\ L\ gaz \rightarrow x = 672\ L\ gaz\]</div>

### 6. **Care este volumul gazului de sinteză (c.n), obținut prin oxidarea parțială cu vapori de apă, a 80 g metan?**

<div class="math display">\[80g\ \ \ {CH}_{4}\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots x\ L\ gaz \rightarrow x = 448\ L\ gaz\]</div>

### 7. **Cât acid cianhidric se obține prin amonoxidarea metanului, dacă s -- au folosit 85 g de amoniac?**

### 11. **Ce formulă moleculară are o hidrocarbură care conține 82,76% C și 17,24 % H și are masa molară M = 58 ?**

Rezolvare:

<div class="math display">\[\left\{ \begin{array}{r}
C\ \frac{82,76}{12} = 6,89 \rightarrow \frac{6,89}{6,89} = 1 \rightarrow 2 \\
H\ \frac{17,24}{1} = 17,24 \rightarrow \frac{17,24}{6,89} = 2,5 \rightarrow 5
\end{array} \rightarrow C_{2}H_{5} \right.\ \ formula\ brută\]</div>

<div class="math display">\[n = \frac{M_{fm}}{M_{fb}} = \frac{58}{29} = 2 \rightarrow C_{4}H_{10},\ butan\ este\ formula\ moleculară\]</div>

**ALCHENE**

### 12. **Care este alchena cu masa molară egală cu 42 g ?**

Rezolvare:<span class="math inline">\(C_{n}H_{2n}\ cu\ M = 14n\ \ și\ 14n = 42 \rightarrow n = 3\  \rightarrow C_{3}H_{6}\ ,\)</span>

<div class="math display">\[\ sau\ {CH}_{2} = CH - {CH}_{3},\ propenă\]</div>

### 13. **Ce cantitate de etenă se obține prin cracarea a 220 g propan?**

Rezolvare: <span class="math inline">\({CH}_{3} - {CH}_{2} - {CH}_{3} \rightarrow {CH}_{2} = {CH}_{2} + {CH}_{4}\)</span>

<div class="math display">\[44\ g\ C_{3}H_{8}\ldots\ldots\ldots\ldots\ldots 28\ g\ C_{2}H_{4}\]</div>

<div class="math display">\[220\ g\ C_{3}H_{8}\ldots\ldots\ldots\ldots\ldots x\ g\ C_{2}H_{4} \rightarrow x = 140\ g\ C_{2}H_{4}\ etenă\]</div>

### 14. **Ce cantitate de etenă se obține din 138 g alcool etilic?**

Rezolvare: <span class="math inline">\({CH}_{3} - {CH}_{2} - OH \rightarrow {CH}_{2} = {CH}_{2} + H_{2}O\)</span>

<div class="math display">\[46\ g\ {CH}_{3} - {CH}_{2} - OH\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots..28\ g\ {CH}_{2} = {CH}_{2}\]</div>

<div class="math display">\[138\ g\ {CH}_{3} - {CH}_{2} - OH\ldots\ldots\ldots\ldots\ldots\ldots\ldots\ldots.x\ g\ {CH}_{2} = {CH}_{2} \rightarrow x = 84\ g\ {CH}_{2} = {CH}_{2}(etenă)\]</div>

### 15. **Ce cantitate de propenă se obține prin reacția de eliminare a acidului clorhidric din 785 g 1 -- clor -- propan?**

Rezolvare: <span class="math inline">\(Cl - {CH}_{2} - {CH}_{2} - {CH}_{3} \rightarrow {CH}_{2} = CH - {CH}_{3} + HCl\)</span>

<div class="math display">\[78,5\ g\ Cl - {CH}_{2} - {CH}_{2} - {CH}_{3}\ldots\ldots\ldots\ldots\ldots\ldots\ldots 44\ g\ {CH}_{2} = CH - {CH}_{3}\]</div>

<div class="math display">\[785\ g\ Cl - {CH}_{2} - {CH}_{2} - {CH}_{3}\ldots\ldots\ldots\ldots\ldots\ldots\ldots x\ g\ {CH}_{2} = CH - {CH}_{3}\]</div>

<div class="math display">\[\rightarrow x = 440\ g\ {CH}_{2} = CH - {CH}_{3}\ (propenă)\ \]</div>

### 16. **Ce cantitate de etan se obține prin hidrogenarea a 84 g etenă?**

<div class="math display">\[{84\ g\ CH}_{2} = {CH}_{2}\ldots\ldots\ldots\ldots x\ g\ {CH}_{3} - {CH}_{3} \rightarrow x = 90\ g\ {CH}_{3} - {CH}_{3},\ etan\ \]</div>

### 18. **Ce cantitate de 2 -- clorpropan se obține prin adiția acidului clorhidric la 176 g propenă?**

### 19. **Ce cantitate alcool etilic se obține prin adiția apei la 140 g etenă?**

### 25. **Care este randamentul reacției de polimerizare a etenei, dacă din 900 g etenă se obțin 540 g polietenă?**
